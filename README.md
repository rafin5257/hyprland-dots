install the basic hyprland, kitty, rofi, waybar.

replace the basic configs from here to your setup and enjoy!


note : if u want the wallpapers working, make a "wallpaper" folder in your home directory and put your wallpapers there.

the wallpaper script will work from waybar!


note: i use capslock for backspace which you can customize in hyprland.conf

just look for caps:backspace in the input section
